# DApp_BlockChain
Block Chain DAPP with Web3.js + Solidity + Truffle + Ganache + MetaMask

## 技术栈
- Remix：在线solidity编译器
- Truffle：智能合约框架
- Ganache：本地区块链开发环境
- MetaMask：浏览器以太坊客户端插件
- DApp：去中心化应用
- Web3.js：以太坊客户端js库
- Ethereum：智能合约公链平台
- Hyperledger Fabric：开源区块链框架

## 实战
- 代币+交易所
- 投票
- 众筹
- nft（数字藏品/元宇宙）
- 模拟挖矿
