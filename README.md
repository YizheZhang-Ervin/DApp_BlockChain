# DApp_BlockChain
Block Chain DAPP with Web3.js + Solidity + Truffle + Ganache + MetaMask

## 基础知识
- 区块链
    - 分布式存储
    - 账本加密算法：hash算法
    - 双花
    - 共识机制
    - P2P通信
    - 去中心化
- 比特币
    - 比特币分叉
        - 硬分叉：bitcoin cash & bitcoin
    - 工作量证明PoW
- 以太坊
    - 以太坊分叉(THE DAO)
        - 硬分叉：ETC & ETH
    - 以太币
    - 权益证明PoS
    - 插件MetaMask：助记词
    - 智能合约
    - ghost协议
    - casper协议(PoW+PoS，含validator)

## 技术栈
- Remix：在线solidity编译器
- Truffle：智能合约框架
- Ganache：本地区块链开发环境
- MetaMask：浏览器以太坊客户端插件
- DApp：去中心化应用
- Web3.js：以太坊客户端js库
- Ethereum：智能合约公链平台
- Hyperledger Fabric：开源区块链框架
