# nextjs

## usage
```
npm create next-app@latest
npm run dev
npm install @rainbow-me/rainbowkit wagmi viem@2.x  @wagmi/core
npm install @tanstack/react-query @radix-ui/react-tabs react-icons
npm install @tailwindcss/postcss postcss
```

npm install

npm test:unit
npm cache
